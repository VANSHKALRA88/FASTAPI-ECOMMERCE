# 🚀 FastAPI CRUD API

A simple and structured FastAPI backend project implementing basic CRUD (Create, Read, Update, Delete) operations for product management.

---

## 📌 Features

- ✅ Create a new product
- ✅ Get all products
- ✅ Get single product by ID
- ✅ Update existing product
- ✅ Delete product
- ✅ JSON-based data storage
- ✅ Clean service-layer architecture
- ✅ Interactive Swagger documentation

---

## 🛠 Tech Stack

- Python
- FastAPI
- Pydantic
- Uvicorn
